name: 불필요한 질문 하지 않기
description: 이미 설정된 인프라에 대해 반복적으로 확인하거나 질문하지 말 것
type: feedback
---

이미 설정/완료된 것에 대해 불필요한 확인 질문을 하지 말 것.
사용자가 "설정되어 있어"라고 하면 바로 다음 단계로 진행.

**Why:** 사용자가 이미 인프라(GitHub Actions, RunPod 엔드포인트 등)를 설정해둔 상태인데 매번 "설정되어 있나요?"라고 물어보면 시간 낭비이고 답답함을 유발.

**How to apply:** 코드 push 후 Docker 빌드, RunPod 배포 등 이미 자동화된 파이프라인이 있다면 당연히 작동 중이라고 가정하고 다음 단계로 바로 안내할 것. 확인이 필요하면 API로 직접 확인하되 사용자에게 질문하지 말 것.
