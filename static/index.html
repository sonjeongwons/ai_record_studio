학습 설정

AI가 목소리를 배울 때 사용하는 세부 옵션입니다. 잘 모르시면 아무것도 건드리지 않고 '학습 시작'을 누르셔도 됩니다! 기본값으로도 좋은 결과를 얻을 수 있습니다. 각 옵션 옆 ⓘ 아이콘을 누르면 자세한 설명을 볼 수 있습니다.

✅ 자연스러운 목소리를 위한 권장 설정
• 데이터 10분 이하: 에폭 100~150, 40kHz
• 데이터 10~25분: 에폭 150~200
• 데이터 25~60분: 에폭 200 (기본값, 권장)
• 스튜디오 녹음 60분+: 에폭 200~300
💡 데이터 품질 최우선: 깨끗한 보컬 10분 > 잡음 있는 보컬 30분.
🎯 최적 데이터량: 30~60분 (다양한 감정/속도/음역 포함 시 가장 자연스러운 결과).
🫁 숨소리 보존: 숨소리를 제거하지 마세요! 숨소리 없는 학습 데이터는 기계음의 원인입니다.
📊 체크포인트 비교: 매 10에폭마다 모델이 저장됩니다. 여러 체크포인트를 청취하여 최적 에폭을 찾으세요.
⚠️ 과학습 방지: 과학습 감지가 내장되어 있어, 자동으로 최적 시점에 멈출 수 있습니다.

RunPod API 설정

RunPod는 AI 학습과 변환에 사용하는 GPU 클라우드 서비스입니다. API Key와 Endpoint ID를 입력해야 학습·변환 기능이 동작합니다.

runpod.io에서 계정을 만들고, Serverless Endpoint를 배포한 후 아래 정보를 입력하세요.

클라우드 스토리지 (R2)

학습된 모델 파일과 오디오를 RunPod 서버로 전송할 때 사용하는 임시 저장 공간입니다. Cloudflare R2는 무료 플랜으로 시작할 수 있습니다.

모델과 오디오 파일 전송에 사용됩니다. Cloudflare 대시보드 → R2 → API 토큰에서 확인하세요. RunPod 템플릿 환경변수와 동일한 값을 입력합니다.

PC 간 작업 동기화

다른 PC에서도 동일한 모델, 전처리 파일, 변환 결과물을 사용할 수 있도록 R2 클라우드에 동기화합니다.

백업: 현재 PC의 DB + 모든 파일을 클라우드에 업로드합니다.
복원: 클라우드에서 DB + 모든 파일을 다운로드하여 이 PC에 세팅합니다.
R2 설정이 먼저 완료되어야 합니다.

· 배치 크기: 기본 8 (43분+ 데이터 권장). 16으로 올리면 더 빨라집니다.
· 과학습 감지: 50에폭 무개선 시 자동 조기 중단되므로 에폭을 높게 설정해도 불필요한 비용이 발생하지 않습니다.
· 실측 참고: 30분 데이터 + Crepe + 500에폭 ≈ 약 6시간, 약 $15 (2026.03 실측).
