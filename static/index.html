학습 설정

AI가 목소리를 배울 때 사용하는 세부 옵션입니다. 잘 모르시면 아무것도 건드리지 않고 '학습 시작'을 누르셔도 됩니다! 기본값으로도 좋은 결과를 얻을 수 있습니다. 각 옵션 옆 ⓘ 아이콘을 누르면 자세한 설명을 볼 수 있습니다.

✅ 자연스러운 목소리를 위한 권장 설정
• 데이터 5분 이하: 에폭 500, 40kHz
• 데이터 5~15분: 에폭 800, 48kHz
• 데이터 15분 이상: 에폭 1000, 48kHz
• 스튜디오 녹음 30분+: 에폭 1500, 48kHz

⚠️ 과학습 방지: 과학습 감지가 내장되어 있어, 설정한 에폭에 도달하기 전에 자동으로 최적 시점에 멈출 수 있습니다.

RunPod API 설정

RunPod는 AI 학습과 변환에 사용하는 GPU 클라우드 서비스입니다. API Key와 Endpoint ID를 입력해야 학습·변환 기능이 동작합니다.

runpod.io에서 계정을 만들고, Serverless Endpoint를 배포한 후 아래 정보를 입력하세요.

클라우드 스토리지 (R2)

학습된 모델 파일과 오디오를 RunPod 서버로 전송할 때 사용하는 임시 저장 공간입니다. Cloudflare R2는 무료 플랜으로 시작할 수 있습니다.

모델과 오디오 파일 전송에 사용됩니다. Cloudflare 대시보드 → R2 → API 토큰에서 확인하세요. RunPod 템플릿 환경변수와 동일한 값을 입력합니다.

💰 과금 안내 — 단계별 비용 & 소요 시간

RunPod GPU 사용 비용과 Cloudflare R2 스토리지 비용에 대한 상세 안내입니다. 각 단계를 시작하기 전에 참고하세요.
